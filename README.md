# Fleet Management System

## Project Description

The Fleet Management System is designed to help businesses manage their fleet of vehicles efficiently. It provides tools for tracking vehicle locations, monitoring fuel consumption, scheduling maintenance, and generating reports.

## Setup Instructions

1. **Clone the Repository**:  
   ```bash
   git clone https://github.com/Giscmr/fleet-management-system.git
   cd fleet-management-system
   ```

2. **Install Dependencies**:  
   Make sure you have Node.js installed. Run the following command to install the required dependencies:
   ```bash
   npm install
   ```

3. **Run the Application**:  
   To start the application, run:
   ```bash
   npm start
   ```

4. **Access the Application**:  
   Open your web browser and navigate to `http://localhost:3000` to view the application.  
   
5. **Database Setup**:  
   Ensure to configure your database settings in the `.env` file. Example:
   ```
   DB_HOST=localhost
   DB_USER=root
   DB_PASS=password
   ```
